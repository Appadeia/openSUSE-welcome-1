There are certain proprietary drivers (such as nvidia) and packages (such as flash) that Korora cannot ship by default. They are however, available for you to install if you want them.

To get drivers such as nvidia (for NVIDIA graphics cards) or fglrx (for AMD/ATI graphics cards), use the Jockey device manager. It is available in the applications menu under the name of Additional Drivers.

Want to install Skype? Then, just head over to http://skype.com to download the package for Fedora and double click to install.
